\doxysubsection{Detailed Description}
Reprezintă un overlay pentru introducerea numelui jucătorului. 

Această stare permite jucătorului să introducă un nume, care este apoi folosit pentru a salva sau încărca progresul jocului. Extinde \mbox{\hyperlink{classgamestates_1_1State}{State}} și implementează \mbox{\hyperlink{interfacegamestates_1_1Statemethods}{Statemethods}}.
