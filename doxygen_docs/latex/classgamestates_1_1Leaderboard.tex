\doxysubsection{Detailed Description}
Reprezintă starea de joc pentru afișarea clasamentului (\mbox{\hyperlink{classgamestates_1_1Leaderboard}{Leaderboard}}). 

Încarcă datele jucătorilor din baza de date, le sortează după scor și le afișează într-\/o listă derulabilă. Extinde \mbox{\hyperlink{classgamestates_1_1State}{State}} și implementează \mbox{\hyperlink{interfacegamestates_1_1Statemethods}{Statemethods}}.
